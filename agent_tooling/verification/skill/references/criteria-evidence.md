# Evidence and Source Criteria

Score source-backed claims from 0 to 20 for source authority, primary-source status, exact support for the statement, quotation or data fidelity, provenance, currency for the stated artifact revision, and material counterevidence coverage.

The source verifier follows each factual assertion to a primary source or a reproduced source record. The skeptic checks whether the source is secondary, stale, misquoted, superseded, scoped differently, or contradicted by a more authoritative record. Store durable locations and the current artifact revision in the ledger.

Only current `primary_source` or `reproduced_source` evidence closes an evidence, research, source, or general claim. `LLM_SUPPORTED` is triage-only. A requested closure with an LLM summary, unsourced citation, or inaccessible source is `INCONCLUSIVE` with the required source recorded as an open obligation.
