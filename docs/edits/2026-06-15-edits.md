## Fixes applied

- **Fix 2 (medium, INFO-1) — Laplace natural gradient rode the Gaussian Fisher.** Made the Fisher
  preconditioner FAMILY-KEYED (the one place a non-Gaussian family had no pure path). New
  `BeliefParams.natural_gradient` hook (`families/base.py`, base raises so a new family must declare its
  metric); `DiagonalGaussian`/`FullGaussian` delegate to the pinned `retraction.natural_gradient`
  (byte-identical, local import — no families↔geometry cycle); `DiagonalLaplace` returns `b^2 * grad` on
  BOTH coords (Laplace `I_mu = I_b = 1/b^2`, sympy-verified — the Gaussian metric mis-scaled the mean by a
  state-dependent `1/b`). `e_step.py` dispatches via `get_family(family)(...).natural_gradient(...)`;
  removed the now-orphaned direct import.
- **Fix 1 (medium, CR-2/PY-2/INFO-2/IMPL-1) — Laplace decoded as Gaussian, silently.** `config.py`:
  UserWarning when `use_prior_bank=True` and `family not in {gaussian_diagonal, gaussian_full}` (the decode
  kernels hardcode Gaussian KL). Warning, not a new kernel — the pure readout paths already exist (a
  Gaussian family, or `use_prior_bank=False`'s family-agnostic linear decode). Mirrors the existing
  renyi/alpha decode-mismatch warning.
- **Fix 3 (medium, TFM-1) — T5 bucket math unvalidated.** `config.py`: raise when
  `t5_max_distance <= t5_num_buckets // 2` (the log-bucketing denominator → 0 or negative → IndexError or
  wrong-end bucket). Defaults 128/32 safe; covers bidirectional (`nb//4`) a fortiori.
- **Fix 4 (medium, DBG-006) — t5_bias froze silently under straight_through.** `model.py`: extended the
  freeze warning from `=='detach'` to `in ('detach','straight_through')` (straight_through detaches the
  per-iteration belief tangent, t5_bias's only gradient carrier) and corrected the message, which had
  wrongly advised straight_through as a fix.
- **Fix 5 (medium, DBG-002) — s_e_step refines a non-Gaussian belief as Gaussian.** `config.py`: warn (not
  raise — the mixed prior/posterior is well-posed, no NaN) when `s_e_step=True` and
  `family != 'gaussian_diagonal'`.
- **Fix 6 (medium, REF-2) — _refine_s drops RoPE.** `model.py`: documented the intentional RoPE omission
  for the s-channel (mirrors the existing `transport_mode='flat'` note); the model channel is
  position-coupled through gamma attention only. Doc-only (threading RoPE onto the model channel is a
  modeling choice, not silently added).
- **Fix 7 (medium, PERF-06) — full-cov sandwich re-cast oh.double() H² times.** `transport.py`
  `_factored_full_covariance`: store `omega_h.double()` in the blocks list (H casts) instead of re-casting
  `oh`/`oh2` inside the H×H inner loop. Value-identical (held by `test_fullcov_alpha_roadmap`).
- **CR-1 (low) — t5_learnable_bias inert with no t5 channel.** `model.py`: warn that the toggle is inert
  when no `t5_relative_bias` channel is active.
