## Headline: the theory core is verified-sound

The math/theory experts returned essentially clean, verified symbolically (SymPy) and numerically,
not by reading comments:

- **Information geometry — no findings.** Diagonal-Gaussian Fisher and natural gradient
  (`nat_mu=sigma·grad_mu`, `nat_sigma=2 sigma^2·grad_sigma`) are the exact metric inverse; Rényi(α=1)
  closed form is exactly KL in the limit; the `alpha*=c0/(b0+D)` envelope cancellation holds; the
  `tau·beta·log(beta/pi)` term makes `beta=softmax(-E/tau+log pi)` the exact stationary point;
  non-negativity / self-divergence-zero verified.
- **Transformer/attention — no findings.** Causal mask admits exactly `j<=i`; `causal_alibi` is the
  faithful per-head Press slope schedule on the causal triangle; `tau=kappa·sqrt(d_head)` applied
  per-block (not sqrt(K)); softmax stable and over the key axis; BCH positional phi preserves
  `Omega_ii=I` so it cannot break causality.
- **Variational/free-energy — one low diagnostics-only item** (metrics.py:148). E/M separation, the
  envelope identity (kernel == filtering-oracle to ~2e-7), and no-label-leakage all verified; the
  s-channel terms enter the loss exactly once under `s_e_step=True` (no double-count).
- **Gauge theory — flat block_glk pure path sound.** Cocycle consistency (`Omega_ii=I`,
  `Omega_ij Omega_jk=Omega_ik`), `embed_phi`/`extract_phi` exact inverse, 2800 block-diagonal
  full-rank generators, gauge-invariant energy, and the head-mixer equivariance footprint all hold.
- **Differential geometry — active SPD/exp-log/transport correct.** Affine retraction matches the
  exact `exp_Sigma`; diagonal sandwich `sum_l Omega_ijkl^2 sigma_jl == diag(Omega Sigma Omega^T)`;
  factored vs dense match to 1e-10.

## Surviving Punch List (confirmed, ranked; no criticals)

No critical findings. No correctness defect on the active path (the two scary candidates were
probe-refuted; the math core is verified). The surviving items are modularity, robustness,
performance, and latent guards.

**Medium**
1. **Registry-bypass validation** — `phi_precond_mode`, `phi_retract_mode`, `pos_phi_compose`
   validated against hardcoded literal tuples, not their live registries (`_PRECOND`, `_COMPOSE`).
   `config.py:23-25, 764, 783, 786`. Violates the CLAUDE.md "add a variant by registering, never by
   editing call sites" contract; a newly-registered preconditioner/composer is silently rejected.
   *Fix:* validate against `tuple(sorted(_PRECOND))` / `tuple(sorted(lie_ops._COMPOSE))`.
2. **`e_step` `**kwargs` silently swallows misspelled config keys** — `e_step.py:517,547` forwards
   `**kwargs`; a typo'd knob is a silent no-op (unlike `free_energy_value`, which rejects unknown
   kwargs). A real footgun for a config-driven project. *Fix:* enumerate explicit kwargs or validate
   the key set against `e_step_iteration`'s parameters.
3. **Per-forward dense `(B,N,K,K)` zero-allocs on the active path** — `_from_equal_diag_blocks`
   (`lie_ops.py:216`, every BCH compose) and `_blockwise_matrix_exp` (`transport.py:350`, twice per
   transport) each allocate and zero a full `(B,N,K,K)` (~160 MB at the active K=140,B=16,N=128),
   saved for backward, though only the `H` `d×d` diagonal blocks are nonzero (`transport.py:350`
   overlaps prior DA-P1c). *Fix:* keep the compact `(B,N,H,d,d)` block representation; value-identical.
4. **`_refine_s` recomputes `exp(r_sigma_log).clamp(...)` on frozen params every forward** —
   `model.py:418-419`, live under `s_e_step=True`. *Fix:* cache the frozen `r_sigma` as a buffer.
5. **`safe_kl_clamp` masks NaN→kl_max with zero gradient, no NaN-rate monitor wired** —
   `families/base.py:26-27`; `numerics.py` monitors exist but are never called on the hot path, so a
   real Cholesky/eigh blowup is indistinguishable from a saturated-but-finite divergence. *Fix:* wire
   an opt-in (default-off) NaN-rate monitor before `nan_to_num`.
6. **`pos_phi_free` trains with no `max_norm` bound** — `positional_phi.py:139` composes
   `pos_phi_free` with no norm clamp, while belief phi gets `max_norm=5` in the retraction; transport
   `cond(Omega)~exp(2‖phi‖)`. Latent at the small init, but the trained table is unbounded. *Fix:*
   clamp the composed phi or add a norm penalty on `pos_phi_free`.

**Low**
7. **`invariant_families` metadata mismatch** — `groups.py:40` declares `("gaussian",)` but the
   family registry keys are `"gaussian_full"`/`"gaussian_diagonal"`, so `invariant_for` returns False
   for both real families. Latent (test-only consumer today). *Fix:* declare `("gaussian_full",)` and
   alias the `"gaussian"` prefix.
8. **`t5_num_buckets=1` → ZeroDivisionError** — `attention_prior.py:215` (`max_exact=nb//2=0`);
   `config.py:680` validates only `>= 1`. *Fix:* validate `>= 2` (or guard `max_exact=max(1,nb//2)`).
9. **`attention_tau` 0-d tensor kappa guard hole** — `free_energy.py:53-54` only checks `dim()==1`,
   so a 0-d wrong-intent kappa broadcasts silently. *Fix:* also handle `dim()==0`.
10. **Dead code (CLAUDE.md: mention, don't delete pre-existing).** `_VALID_ENCODE_MODES` /
    `_VALID_DECODE_MODES` unused (`config.py:20-21`); `run_training` + its private `_banner`
    deprecated with zero callers (`train.py:629-693`; also silently selects CPU — known DA-P3a);
    `decode_log_scale` allocated but unread under `use_prior_bank=False` (`prior_bank.py:135`).
11. **`metrics.py:148` per-head tau broadcast** — uses bare `tau` instead of `_broadcast_tau`;
    diagnostics-only and so_n/sp_n-only (block_glk tau is scalar). *Fix:* use `_broadcast_tau`.
12. **Type-contract drift** — `_as_coeff` untyped (`block.py:19`); `_retract_core` annotates
    `trust_region`/`max_norm` as `float` but tests for `None` (`lie_ops.py:326`); `b0`/`c0` annotated
    `float` but receive `Tensor` (`e_step.py:160`). Cosmetic; mypy-strict drift.

**Latent (real on their toggle, not the active path)**

- `decode_ce_diagonal_chunked` recomputes `1/sigma_v_all` and `log(sigma_v_all).sum` per chunk per
