# Blue Panel Choice - Phase 3 Rebuttal

Selected panel: variational, transformer-ml, implementation-engineer, numerical-analyst, philosophy-of-science.

- variational: Red's strongest attack is semantic collapse from expected free energy into a generic uncertainty reranker, so the defense needs the active-inference functional and its limiting cases pinned down.
- transformer-ml: The rebuttal must face decoding and reranking baselines directly, including matched candidate pools, beam or truncation sampling, and length-normalized log-probability controls.
- implementation-engineer: The claim is only defensible if the V3 code has a finite inference-time hook and if the current blockers are stated without pretending the training path already exposes an active-inference policy model.
- numerical-analyst: The empirical value of the scorer depends on compute budgets, seed noise, ablation design, and whether sigma and ambiguity terms have causal force.
- philosophy-of-science: Mandatory frame-checking expert; the rebuttal must concede what is not established and state conditions under which the proposed scorer is just a failed auxiliary hypothesis.

Discounted theory roster: geometer, info-geometer, and gauge-theorist are less central for this rebuttal because Red's attack is not that GL(K) geometry is impossible; it is that policy semantics, language-model baselines, code reachability, and falsification discipline are under-specified.
