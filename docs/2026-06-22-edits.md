## EXP-3 — Sigma_q as calibrated Fisher uncertainty

- `vfe3/viz/extract.py`: new `belief_ce_bank(model, loader, ...)` — the Sigma_q<->CE **join**.
  Per `(tokens, targets)` batch it runs `model(tokens)` for decode CE and replays the belief stack
  (`vfe_stack`, same head_mixer/cg_coupling/block_norm as the forward) for `out.sigma`, then returns
  aligned per-token `{tr_sigma, ce, token_ids}` masked to `targets != -100`. `tr_sigma[n]` is the
  belief that produced `logits[n]` predicting `targets[n]`, so the pair is position-aligned.
- `vfe3/viz/figures.py`: three registered figures — `reliability_diagram` (decode acc-vs-conf
  control, consumes the existing `research.json` reliability bins), `sigma_stratified_error` (mean CE
  per tr(Sigma_q) decile + bootstrap 10-90 band), `sigma_ce_scatter` (tr(Sigma_q)-vs-CE hexbin with
  Spearman rho). Helper `_spearman_np`.
- `vfe3/viz/report.py`: `generate_figures` now builds the `belief_ce_bank` input, reads the
  `research.json` reliability bins, and emits the three figures (best-effort `_emit`, like the rest).
- `vfe3/run_artifacts.py`: `_write_research_artifacts` persists the headline scalars
  `sigma_ce_spearman`, `sigma_trace_cv`, and the `sigma_trace_cv_gate_pass` (CV > 0.10) flag, off the
  hot path, each guarded.

## EXP-4 — canonical-F vs entropy-suppressed surrogate (C1): the −τ⁻¹Cov_β gap

The `attention_entropy` sweep already ran canon-vs-surrogate PPL; the remaining build was the direct
measurement of the −τ⁻¹Cov_β(E,∇E) attention-entropy gradient gap + its figures.

- `vfe3/viz/extract.py`: new `attention_entropy_cov_gap(model, token_ids)` — builds the converged
  belief q*, then differences the autograd **oracle** gradient with `include_attention_entropy=True`
  (canonical) vs `False` (surrogate) on the SAME snapshot; their difference is exactly the entropy
  term's gradient = −τ⁻¹Cov_β(E,∇E) (the envelope identity, pinned synthetically in
  `test_free_energy.py`). Returns `cov_gap_per_token` (N,) + the scalar `cov_gap`. The oracle is used
  for both arms because the closed-form kernel never computes the entropy term.
- `ablation.py`: `_cell_diagnostics` now emits a guarded `cov_gap` (CSV column); the
  `attention_entropy` sweep is the **2×2 entropy × κ** grid (canon/surr × κ∈{1.0, 0.25}, all
  `oracle_unroll_grad=True`); new `_plot_attention_entropy` driver (PPL-gap grouped bar + cov-gap-vs-κ
  line) dispatched per sweep (no-op when absent).
- `vfe3/viz/figures.py`: `entropy_ppl_gap` (canonical vs surrogate PPL by κ, gap annotated) +
  `cov_gap_vs_kappa` (‖−τ⁻¹Cov_β‖ vs κ, one series per arm).

## How to run

- EXP-3: train a checkpoint, then figures auto-emit via `generate_figures` (or `make_figures.py`);
  `research.json` carries `sigma_ce_spearman` / `sigma_trace_cv` / gate flag.
- EXP-7: `ablation.py` `CONFIG["sweep"]="rho_handoff"` -> per-sweep `rho_handoff_rank_collapse.png`.
- EXP-5: `scaling.py` `CONFIG["routes"]=["infer_T"]` (sweeps n_e_steps via finalize_run), then
  `python scaling_analysis.py` -> `estep_capacity.png` + `f_ce_decorrelation.png` + the two Pearsons.
- EXP-4: `ablation.py` `CONFIG["sweep"]="attention_entropy"` -> per-sweep `attention_entropy_ppl_gap.png`
  + `attention_entropy_cov_gap.png` (and the `cov_gap` column in `sweep_results.csv`).
- EXP-8: `ablation.py` `CONFIG["sweep"]` in {`gauge_mstep_optim`, `m_phi_lr_natgrad`, `mass_phi`} ->
  per-cell metrics.csv carries `cos_nat_phi` / `pullback_cond_*` / `wall_clock_s`; per-sweep
  `<sweep>_wallclock_convergence.png` (+ the LR bowl from `_plot_one_sweep` on `m_phi_lr_natgrad`).
- EXP-2/9/10/11: `ablation.py` `CONFIG["sweep"]` in {`gauge_transport`, `gauge_equivariance`,
  `cg_coupling`, `kappa_beta_per_head`} -> the per-sweep figure auto-emits (gauge bars / residual
  drift / PPL+equivariance / κ dispersion).
- EXP-6: `python scaling_analysis.py` after the `grow_K` + `grow_K_mup` routes -> `kmup_stability.png`.
- EXP-1: `python multiseed_analysis.py` after a `NUM_RUNS`/`SEEDS` run -> `figures/ppl_noise_band.png`.
